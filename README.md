# 📦 Sistema de Estoque

Sistema simples de gerenciamento de estoque desenvolvido em Python. Permite cadastrar, listar, buscar, alterar e remover produtos, com persistência em JSON.

## 🚀 Funcionalidades

- ✅ Adicionar produtos (nome, quantidade, preço)
- ✅ Listar todos os produtos
- ✅ Buscar produtos por nome
- ✅ Alterar produtos (nome, quantidade ou preço)
- ✅ Remover produtos
- ✅ Validações de entrada (nomes, números negativos)
- ✅ Persistência em arquivo JSON
- ✅ Interface via terminal (CLI)

## 🛠️ Tecnologias utilizadas

- Python 3
- JSON (para persistência dos dados)
- Tratamento de erros com try/except
